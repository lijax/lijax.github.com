Tools I used for markdown writing

Date Sun 03 March 2013 By Yanbo Ye Category Computer Work. Tags programing markdown Pandoc Sublime Text

Editors

Sublime Text and plugins on Linux

Sublime Text is an awesome text editor supporting almost all programming languages. It also support Windows and Mac including Linux. Some useful functions users love are Goto Anything, Multiple Selections, Command Palette and Split Editing, etc. While the most fantastic feature is its easy and elegant plugin system using JSON and python language and its open source plugin community.

So it can support markdown with highlight and there are some plugins make markdown writing much easier.

List of markdown plugins:

  • Markdown Preview: preview in browser.
  • Markdown Slideshow: markdown slideshow to Html5 slideshow.
  • Pandoc: Pandoc integration in Sublime.
  • Pandown: Another Pandoc integration with more output support.
  • Pandoc Academic: Another Pandoc integration with bibliography support.

Editors on other platforms

  • Windows: MarkdownPad
  • Chrome: MaDe
  • Android: Writer with Dropbox syncing.

Pandoc

Pandoc is a general markup converter. It can convert from one markup format to another and to DOCX, PDF, EPUB document. The blow network shows the supported markup formats and conversions. Pandoc allow you to write document in markdown and release it to any other format.
